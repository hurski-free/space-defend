# Webpack + TypeScript + ESLint Starter

## Setup

1. Install dependencies:

   npm install --save-dev webpack webpack-cli webpack-dev-server typescript ts-loader eslint @typescript-eslint/parser @typescript-eslint/eslint-plugin eslint-webpack-plugin

2. Run development server:

   npm run dev

3. Build for production:

   npm run build

4. Run lint:

   npm run lint
